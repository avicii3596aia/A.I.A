# A.I.A - Avicii Integrated Analysis

## Overview
This is the main entry point for the Avicii Integrated Analysis system - a comprehensive MATLAB GUI toolkit for biological data analysis.

## Features
- **PlaterReader**: Bacterial growth analysis (OD and Luciferase measurements)
- **Image Analysis**: Pellicle, bacterial growth, pore analysis, and Sanger sequencing visualization
- **Advanced Analysis**: Dose-response, colony counting, biofilm analysis
- **Microscopy Tools**: Cell counting and viability analysis
- **Genomics & Automation**: Genomics tools and batch processing
- **Calculators**: Molarity and media preparation calculators

## How to Use
1. Run `AIA_v4_main.m` in MATLAB
2. A GUI window will open with different analysis options
3. Click any button to launch the specific analysis module
4. Follow the on-screen instructions

## Requirements
- MATLAB (R2019b or later recommended)
- Image Processing Toolbox (for image analysis features)
- Curve Fitting Toolbox (for dose-response analysis)

## Modules
This main script calls several modules:
- `growth_analysis_module.m`
- `image_analysis_module.m`
- `advanced_analysis_module.m`
- `calculators_module.m`
